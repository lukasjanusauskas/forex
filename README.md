### Overview

This project is based around forex model.
A model predicts a set of foreign exchange rates and displays the predictions on an online dashboard.

### Data:
1. PPP (OECD code: PPP)
2. Exchange rates (https://data.ecb.europa.eu/main-figures/ecb-interest-rates-and-exchange-rates/exchange-rates)
3. GDP ()
4. CPI (OECD API)
5. Stock market growth (later...)
6. Income growth (https://sdmx.oecd.org/public/rest/data/OECD.WISE.INE,DSD_WISE_IDD@DF_IDD,1.0/.A.EAR+INC_DISP..._T.METH2012.D_CUR.?startPeriod=2010&dimensionAtObservation=AllDimensions)
